Те которые внутри?

Для заданных двух массивов строк a1 и a2 возвращается отсортированный массив r в лексикографическом порядке строк a1, которые являются подстроками строк a2.

Пример 1:

a1 = ["arp", "live", "strong"]

a2 = ["lively", "alive", "harp", "sharp", "armstrong"]

вернуть: ["arp", "live", "strong"]

Пример 2:

a1 = ["tarp", "mice", "bull"]

a2 = ["lively", "alive", "harp", "sharp", "armstrong"]

вернуть: []

    Замечания:
  • Массивы записываются в "общей" нотации. Примеры на вашем языке приведены в разделе "Ваши тестовые примеры".
  • В Shell bash a1 и a2 являются строками.
  • Возвращаемое значение представляет собой строку, в которой слова разделены запятыми.
  • Будьте осторожны: в некоторых языках r должен быть без дубликатов.
